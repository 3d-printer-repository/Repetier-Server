@PROJECT_NAME@

Short introduction

@PROJECT_NAME@ is a daemon managing your 3d printers. It gets started automatically by calling the corresponding website. To get started call http://localhost:@SERVER_PORT@ after the installation is finished. The server listens to all IPs, your computer is bound to!

When using the server, it will store several files needed for working in the following directory:
/Library/Application Support/@PROJECT_DIR@ Please do not touch these files, if you are not told in a documentation or a support answer. Changing the files can cause the server to crash or do unexpected things.

Features

  • Run one or more connected printer at once.
  • Autodetect, if a printer goes online.
  • Access your printer from everywhere. You only need a internet connection to this server.
  • Configure access for different users or grant full access to anyone.
  • Watch your printer over a MJPG capable webcam.
  • Precise printing time prediction.
  • Easy configuration. All parameter can be changed from the web interface.
  • Small memory print. Prints 1GB file with only a few MB memory usage.
